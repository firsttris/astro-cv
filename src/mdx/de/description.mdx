# Karrierezusammenfassung

<Divider />

Engagierter Softwareentwickler mit über 10 Jahren Erfahrung in der Entwicklung von zuverlässigem und wartbarem Code für kritische Unternehmensanwendungen. 

Entwickler von <a href="https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner" target="_blank"><Icon name="vscode-icons:file-type-jest" style={{ marginBottom: '-2px'}}/> Jest Runner</a>, einer VSCode-Erweiterung zur Testausführung, die <a href="https://github.com/nrwl/nx/issues/3868" target="_blank">seit 2020</a> als empfohlene Erweiterung von <a href="https://nx.dev/" target="_blank"><Icon name="vscode-icons:file-type-light-nx" style={{ marginBottom: '-2px'}}/> NX</a> installiert wird.

Umfangreiche Erfahrung in moderner Frontend-Entwicklung und Architektur, um skalierbare Benutzeroberflächen effizient zu erstellen und zu warten.

Langjährige Projekterfahrung in der Entwicklung moderner Benutzeroberflächen mit <a href="https://react.dev/" target="_blank"><Icon name="vscode-icons:file-type-reactts" style={{ marginBottom: '-2px'}}/> React</a>, <a href="https://www.typescriptlang.org/" target="_blank"><Icon name="vscode-icons:file-type-typescript-official" style={{ marginBottom: '-2px'}}/> TypeScript</a>.

Spezialisiert auf Testautomatisierung und testgetriebene Entwicklung (TDD).

Vertreter von <a href="https://www.youtube.com/watch?v=UjhX2sVf0eg" target="_blank">Clean Code</a> und <a href="https://wiki.archlinux.de/title/KISS-Prinzip" target="_blank">KISS (Keep it simple and stupid)</a>, ein Prinzip für einfache und effektive Softwareentwicklung.

# Karriere Zusammenfassung

<Divider />

Engagierter Softwareentwickler mit über 10 Jahren Erfahrung in der Entwicklung von robustem Code für kritische Unternehmensanwendungen. 

Entwickler von <a href="#openSource"><Icon name="vscode-icons:file-type-jest" style={{ marginBottom: '-2px'}}/> Jest Runner</a>, einer VSCode-Erweiterung zur Testausführung, die [seit 2020](https://github.com/nrwl/nx/issues/3868) als empfohlene Erweiterung von [<Icon name="vscode-icons:file-type-light-nx" style={{ marginBottom: '-2px'}}/> NX](https://nx.dev/) installiert wird.

Umfangreiche Erfahrung in moderner Frontend-Entwicklung und Architektur, um skalierbare Benutzeroberflächen effizient zu erstellen und zu warten.

Langjährige Projekterfahrung in [<Icon name="vscode-icons:file-type-reactts" style={{ marginBottom: '-2px'}}/> React](https://react.dev/), [<Icon name="vscode-icons:file-type-typescript-official" style={{ marginBottom: '-2px'}}/> TypeScript](https://www.typescriptlang.org/) und [<Icon name="devicon:materialui" style={{ marginBottom: '-2px'}}/> Material UI (MUI)](https://mui.com/).

Spezialisiert auf Testautomatisierung und testgetriebene Entwicklung (TDD).
